# Agente 1: Frontend / Home & Estructura
**Jurisdicción**: `/Frontend/home`

## Responsabilidades
- Mantenimiento de la Landing Page (`index.astro`).
- Layouts globales y componentes compartidos de UI (`Header`, `Footer`, `BottomNav`).
- Diseño visual y consistencia de estilos (`styles/global.css`).
- SEO y Metadatos.

## Estándares
- Usar Tailwind CSS para todo el styling.
- Los componentes deben ser Mobile-First.
- Importar estilos globales desde `@home/styles/global.css`.
